# random-password-generator-python
A simple command-line based random password generator built using Python.
This program allows users to generate secure and customizable passwords with a specified length.

## Features
- Generates strong random passwords
- Includes:
  - Lowercase letters (a–z)
  - Uppercase letters (A–Z)
  - Digits (0–9)
  - Special characters (!, @, #, etc.)
- User-defined password length
- Easy to use command-line interface

## How It Works
- The program prompts the user to enter the desired password length.
- It combines:
  - Lowercase letters
  - Uppercase letters
  - Numbers
  - Special symbols
- It randomly selects characters based on the given length.
- It prints the generated password.
